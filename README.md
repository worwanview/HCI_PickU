# HCI_PickU
โปรเจกต์นี้มุ่งเน้นการ ออกแบบประสบการณ์ผู้ใช้ (UX) สำหรับแอปรับหิ้วสินค้าเพื่อนักศึกษา โดยมีขั้นตอนการดำเนินงานดังนี้

⭐️ วิเคราะห์ความต้องการของผู้ใช้จากการทำ User Research เพื่อเข้าใจปัญหา พฤติกรรม และความต้องการของนักศึกษา

⭐️ กำหนด User Flow เพื่อแสดงลำดับขั้นตอนการใช้งานของผู้ใช้ในระบบ

⭐️ ออกแบบ Usability Test Tasks เพื่อใช้ประเมินความเข้าใจและประสิทธิภาพในการใช้งาน

⭐️ ทดสอบการใช้งานกับผู้ใช้จริง และรวบรวมผลการทดสอบเพื่อนำมาวิเคราะห์ Pain Points

⭐️ นำผลการทดสอบมาปรับปรุงการออกแบบให้เหมาะสมกับผู้ใช้มากขึ้น
